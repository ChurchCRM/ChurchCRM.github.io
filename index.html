A Free Web-Based CRM System for Your Church

With Church Web CRM, you can manage your church's congregation directory, Sunday School, Volunteers, and pledges.

Mobile Friendly

Accessible from anywhere using any modern browser! Smart Phones, iPads, Desktops, or Laptops - The choice is yours.

Always in Development

Church Web CRM is a branch off of two former open source platforms. About 10 years ago, a program called InfoCentral was created and, after a few upgrades, was soon renamed ChurchInfo.

Runs on LAMP

Since ChurchCRM is a web-based application, you don't need to worry about what devices your users use to access it. All the heavy lifting is one on the server using what's commonly known as a "LAMP Stack." LAMP represents a set of applications, along with an operating system: Linux, Apache, MySQL, and PHP. If your church has an existing web site, chances are good that you're already using a LAMP stack. Since ChurchCRM only requires LAMP, you can co-host the application on the same servers as your webpage (often with no additional cost)!
