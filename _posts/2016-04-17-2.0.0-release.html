--- layout: post title: ChurchCRM 2.0.0 Released is available --- On behalf of the team, I am happy to announce the general availability of the Church CRM in version 2.0.0. You can download it from GitHub Releases. The reference documentation is available here. Summary of the release notes are bellow.

What’s New in ChurchCRM 2.0.0

  • 1st ChurchCRM Release
  • Restyle The ChurchInfo web pages
  • Mobile Friendly
  • Added Home Screen Dashboard
  • Added Sunday School Support
  • Added MailChimp export report
  • Display Gender/Age default member photos
  • Search By Check Number
  • Use of gravatar for member photos
  • Global search box on every page
  • Member action buttons
  • Family action buttons
  • Seed data for families

Contributions

Without the community we could not be the successful project we are today. I’d like to thank everyone that created issues & provided feedback.

Feedback Please

If you have feedback on this release, I encourage you to reach out via GitHub Issues, or via the dev chat. Of course the best feedback comes in the form of contributions. Visit the change log to view the full changes.
